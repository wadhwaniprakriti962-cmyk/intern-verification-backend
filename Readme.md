# Intern Verification System - Backend

Backend API for the Intern Verification System.

Backend URL: https://intern-verification-backend.vercel.app

## 📌 Features

- HR Authentication
- JWT Authorization
- Intern Data APIs
- MongoDB Integration
- Secure Password Hashing
- Protected Routes

## 🛠 Tech Stack

- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT
- bcryptjs
- CORS

## 📂 Folder Structure

server/
├── controllers/
├── models/
├── routes/
├── middleware/
├── config/
└── server.js
